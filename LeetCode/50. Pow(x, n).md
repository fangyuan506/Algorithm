# 50. Pow(x, n)

## 题目

题目链接：[50. Pow(x, n) - 力扣（LeetCode）](https://leetcode.cn/problems/powx-n/description/)

题目难度：中等

题目标签：数学

题目详情：

实现 [pow(*x*, *n*)](https://www.cplusplus.com/reference/valarray/pow/) ，即计算 `x` 的整数 `n` 次幂函数（即，`xn` ）。

**示例 1：**

```
输入：x = 2.00000, n = 10
输出：1024.00000
```

**示例 2：**

```
输入：x = 2.10000, n = 3
输出：9.26100
```

**示例 3：**

```
输入：x = 2.00000, n = -2
输出：0.25000
解释：2-2 = 1/22 = 1/4 = 0.25
```

**提示：**

- `-100.0 < x < 100.0`
- `-231 <= n <= 231-1`
- `n` 是一个整数
- 要么 `x` 不为零，要么 `n > 0` 。
- `-104 <= xn <= 104`



## 快速幂

``` java
class Solution {
    public double myPow(double x, int n) {
        if (x == 0) return 0;
        long m = n;
        if (m < 0) {
            x = 1 / x;
            m = -m;
        }
        double res = 1;
        while (m > 0) {
            if ((m & 1) == 1) res *= x;
            x *= x;
            m >>>= 1;
        }
        return res;
    }
}
```
